# Changelog

## [1.0.0] - 2026-06-01

### Added
- Core retry engine with Exponential, Linear, Fixed and Custom strategies
- Dead Letter Queue with replay, analyzer and operator notes
- Storage adapters: SQLite, Redis, PostgreSQL, Memory
- Middleware: Express, Fastify, Next.js App Router, Next.js Pages Router, Hono
- Circuit Breaker (PRO)
- Idempotency guard (DuplicateDetector)
- HMAC signature verification for Stripe, GitHub, Generic
- Real-time Dashboard (HTTP + WebSocket)
- React dashboard SPA
- Full TypeScript support with strict mode
- Unit, integration and E2E test suite (vitest)
